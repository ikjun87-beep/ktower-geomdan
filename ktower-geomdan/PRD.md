## 2. 프로젝트 개요 & 목적

**무엇.** 인천 서구 당하동 1237(검단신도시 C3-①-5) 일반상업BL에 들어서는 근생·업무·판매시설 개발사업 **"The K Tower 검단"**의 분양가 적정성과 사업성을 **공개 데이터 교차검증 기반으로 독립 검토**하는 보고서.

**왜.** 시행사 원안(킴앤코 사업계획서)의 수익률 17.3%가 상층 근생까지 1층 근접가를 적용한 가정 위에 서 있어, 실제 시장 층별가로 재산정 시 수익성이 크게 달라질 수 있다. 의사결정 전에 **위험과 적정가를 객관적으로 짚는다.**

**산출물의 역할.** 투자 권유가 아닌 **의사결정 보조 자료**. 수치는 추정을 포함하며, 최종 결정 전 1차 통계·현장 실사·전문가 검토를 전제로 한다.

## 5. 데이터·방법론 원칙

1. **공개 데이터 교차검증** — 네이버부동산·상가114·KB부동산·한국부동산원·금융위·언론보도 등 다중 출처.
2. **추정 vs 실데이터 명시** — 상업·업무시설은 실거래 공개가 드물어 호가·마케팅 자료가 섞임. 둘을 구분 표기.
3. **수치 절대 재계산 금지(보수성)** — 층별 계약평당가 분해표 미확보 구간은 절대 수입을 임의 재산정하지 않고 **% 민감도**로 제시.
4. **출처 전부 부록에 기재** — 모든 인용은 References에 URL/매체로 추적 가능.

## 6. 제작 워크플로우 (표준 — 주먹구구 방지) ⭐

이 프로젝트의 핵심. 앞으로 모든 갱신은 이 순서를 따른다.

```
[1] 조사·검증     공개 데이터 수집 → 추정/실데이터 구분 → 출처 기록
        │
[2] 본문 작성     report.html (섹션별), deck.html (요약) 수정
        │            · 수치·문장은 §10 품질기준 준수
        │
[3] PDF 변환      Chrome headless 로 HTML → PDF 재생성 (§8)
        │
[4] 검수          미리보기 = 실제 출력 일치 확인, 표기 일관성 점검
        │
[5] 버전 커밋     git add → commit (메시지 규칙 §9) → "제N판"
        │
[6] 원격 백업     git push origin main (GitHub)
```

**원칙.** HTML이 원본(Single Source of Truth), PDF는 항상 HTML에서 재생성한 파생물. PDF만 따로 수정하지 않는다.

## 9. 버전 관리 정책

- **판(版) = 의미 단위의 커밋.** 내용 보강/판올림 시 "제N판"으로 명명.
- **커밋 메시지 규칙**: `제목 — 한 줄 요약` + 본문에 무엇을/왜. 끝에 Co-Authored-By.
- **PDF 동기화**: 본문 수정 커밋에는 재생성한 PDF를 함께 포함.
- **현재 로컬**: origin 대비 **미푸시 커밋 존재**(GitHub 백업 필요).

## 10. 품질 기준

1. **수치 불변 원칙** — 디자인·표기 작업은 분양가·수익률·면적 등 **숫자를 절대 변경하지 않는다.**
2. **표기 일관성** — 제품명·단위·판 번호·검토기준일 통일.
3. **출처 추적성** — 모든 주장에 부록 출처 매칭.
4. **추정 라벨링** — 추정치는 "추정/벤치마크"임을 본문에서 명시.
5. **미리보기 일치** — 화면 = 인쇄 PDF 결과 동일(폰트·여백·줄바꿈).

## 11. 남은 작업 / 검증 항목 (보고서 §7 기준)

| 순위 | 검증 항목 | 출처/방법 | 상태 |
|---|---|---|---|
| 1 | 원안 근생 **층별 단가 가정** 확인(상층에 1층가 적용 여부) | 킴앤코 원본 엑셀 | ☐ 미확보 |
| 2 | 업무·근생 **전용률** 확인 후 전용 평당가 환산(선양 49.4% 대조) | 건축도면·분양면적표 | ☐ |
| 3 | 검단 상업·업무 **미분양·공실 1차 통계** | 부동산원 R-ONE, 통계누리, iH | ☐ |
| 4 | 비교사례(선양 등) **호실별 분양가** | 분양사무소 | ☐ |
| 5 | 시공사·신탁사 **책준·신용보강 의향** | 도급·신탁 협의 | ☐ |

> 1·2번(킴앤코 엑셀, 전용률)이 확보되면 추정→실측 전환으로 **제5판** 업그레이드 가능.
